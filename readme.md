 Estudos de Machine Learning

Este repositório reúne meus estudos práticos e teóricos em Machine Learning, com foco em compreender os fundamentos, algoritmos clássicos e técnicas modernas utilizadas em Ciência de Dados e Inteligência Artificial.

O objetivo é servir como um repositório de aprendizado contínuo, com códigos comentados, experimentos, anotações e exemplos práticos.

 Objetivos

Compreender os fundamentos matemáticos do Machine Learning

Implementar algoritmos do zero para fins didáticos

Utilizar bibliotecas populares como NumPy, Pandas, Scikit-learn e Matplotlib

Analisar dados, treinar modelos e avaliar resultados

Consolidar conhecimento para aplicações acadêmicas e profissionais
